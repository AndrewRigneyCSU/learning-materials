Heading component

The button component can be included within other components like this:

{{> @headings }}

This template for this component looks like this:

<h2 class="label">{{ modifier }}</h2>

<blockquote class="{{ class }}">
  <p>{{ text }}</p>
  <cite> {{ cite }}</cite>
</blockquote>

and it therefore expects a set of data to render it that is in the following format:

{
    "modifier": "default",
    "text": "Someone famous once said something really profound. It was probably Einstein. Yeah Einstein.",
    "cite": "Albert Einstein, 2017",
    "label": "Default"
}

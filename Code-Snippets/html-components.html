Tooltips

To utilise tool tips simply add a span to your paragraph of text and add tooltip="Your text goes here" as an attribue to the tag.

Tooltips are underlined with a dotted line. Tool tips can also be applied to links which go out to the world which makes them quite flexible.

Colours

Colours can be applied to elements using classses rather than inline. For most circumstances classes should be applied to the base HTML element, but to add colour inside an element use a SPAN.

    Colours can used to set the colour for:
  • Backgrounds
  • Text
  • Lines, and
  • Special list types
